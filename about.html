About Geo-search

Geo-search is the Capstone Project for #DevCTrainingWithAndela. It is an app that allows user to use basic feature on a Map such as search for the location of a place, view the Weather condition of a place and convert temperature between Celcius and Fahrenheit measurement. Users will be able to share these information on facebook.
